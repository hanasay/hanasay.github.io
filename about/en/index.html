Kaldi, TDNN-F Algorithm

Initially, I used the Kaldi speech training toolkit for modeling, employing the TDNN-F algorithm for training.

Advantages of TDNN-F: During training, the model learns from both past and future context to choose the most likely answer. This helps the model better match the speakerâ€™s intent and semantics when selecting words.

Espnet, End-to-End and Contextual-Block-Conformer

Later, I switched to using Espnet for training.

Unlike Kaldiâ€™s GMM & HMM structure, Espnet adopts an End-to-End framework.

Pros: Neural networks enable the model to learn automatically without manually optimizing each step.
Cons: When results fall short, there are fewer aspects to tweak. It requires a large amount of training data to perform well with an End-to-End structure.

Collaboration with TAIDE

The TAIDE project is a national research initiative led by Professor Tsung-Han Tsai (NCU) and Professor Hung-Yi Lee (NTU), aiming to build a Taiwan-specific ChatGPT.

SARC collaborated with TAIDE starting September 2023 to localize the model. By November 2023, we had fine-tuned LLMs for Taiwanese Hokkien and Hakka, which were released on the TAIDE public beta platform.

More info on TAIDE


Steps to fine-tune an LLM:

  1. Select a base model.
  2. Expand and optimize the tokenizer for the target language.
  3. Feed the model with a large number of articles in the target language to help it develop a natural sense for the language.
  4. Train the model on Q&A data in the target language to teach it how to respond.
  5. Final model is ready.
